Benutzung des Simplex Algorithmus

  1. Zuerst muss man die Bedingungen festlegen:
  2. After the first necessary step, we begin shaping the Standard form
    • creation is only allowed by setting constraints
  3. Followed by creating the Extended Standard Form
    • formed via introduction of Slack/Surplus variables
    • represented by fulfilling the following rules:
      • x Non-negative constraints of all variables
      • x All remaining constraints expressed as equalities
      • x RHS vector being non-negative
  4. Forming the Canonic “Simplex” Tableau
  5. Performing pivot operations
    • choosing the pivot column (via most negative number in the bottom row)
    • choosing the pivot row (via dividing RHS by the row value and choosing the lowest result)
    • making these two into the pivot (or pivoting point) and turning its value to 1
    • making every value “above” and “below” to 0
    • repeating this cycle until there are no negative numbers in the bottom row

Now a little example..

Our business delivers 3 kinds of products, with the price:
  • A: 7€
  • B: 8€
  • C: 10€
Fabrication time (up to maximum of 1000 hours) for each product:
  • A: 2h
  • B: 3h
  • C: 2h
Assembly time (up to maximum of 800 hours) for each product:
  • A: 1h
  • B: 1h
  • C: 2h

Solving process:

What needs to be done first, is determining the to-be-optimized scenario and constraints, which leads us to the following Standard Form:

7x1+8x2+10x3→max

2x1+3x2+x3≤1000

x1+x2+2x3≤800

From there, we can continue to create the Extended Standard Form, by fulfilling the following rules:
  1. Non-negative constraints of all variables
  2. All remaining constraints expressed as equalities
  3. RHS vector being non-negative

-7x1-8x2-10x3+p=0

2x1+3x2+2x3+S1=1000

x1+x2+2x3+S2=800

After that, we form the Canonic „Simplex“ Tableau and start performing the pivot operations.
  1. pivot column (most negative bottom row number) -> -10, column number 3
  2. pivot row (lowest result of RHS divided by row value) -> 2, row number 2
    • 1000:2=500
    • 800:2=400
    • *red always indicates pivot values*
    x1 x2 x3 s1 s2 p
    s1 2 3 2 1 0 0 1000
    s2 1 1 2 0 1 0 800
    -7 -8 -10 0 0 1 0
  3. Change the pivot row value to 1 via 12R2→R2
    x1 x2 x3 s1 s2 p
    s1 2 3 2 1 0 0 1000
    s2 12 12 1 0 12 0 400
    -7 -8 -10 0 0 1 0
  4. Transforming everything above and below the 1 to 0 first via -2R2 + R1 → R1
    x1 x2 x3 s1 s2 p
    s1 1 2 0 1 -1 0 200
    s2 12 12 1 0 12 0 400
    -7 -8 -10 0 0 1 0
  5. And then via 10R2 + R3 → R3
  6. Replacing the pivot row variable with the pivot column variable
  7. Repeating the whole process by choosing a new pivot column and row
    x1 x2 x3 s1 s2 p
    s1 1 2 0 1 -1 0 200
    x3 12 12 1 0 12 0 400
    -5 -3 0 0 5 1 4000
  8. 12R1→R1
    x1 x2 x3 s1 s2 p
    s1 12 1 0 12 -12 0 100
    x3 12 12 1 0 12 0 400
    -5 -3 0 0 5 1 4000
  9. -12R1+R2→R2
  10. 3R1+R3→R3
    x1 x2 x3 s1 s2 p
    x2 12 1 0 12 -12 0 100
    x3 14 0 1 -14 34 0 350
    -12 0 0 32 72 1 4300
  11. Again, repeating the whole pivot operations process
    x1 x2 x3 s1 s2 p
    x2 12 1 0 12 -12 0 100
    x3 14 0 1 -14 34 0 350
    -12 0 0 32 72 1 4300
  12. 2R1→R1
    x1 x2 x3 s1 s2 p
    x2 1 2 0 1 -1 0 200
    x3 14 0 1 -14 34 0 350
    -12 0 0 32 72 1 4300
  13. -14R1+R2→R2
  14. -12R1+R3→R3
    x1 x2 x3 s1 s2 p
    x1 1 2 0 1 -1 0 200
    x3 0 -12 1 -12 1 0 300
    0 1 0 2 3 1 4400
  15. Look at the LHS variables and set them to RHS values
    • x1 = 200
    • x3 = 300
  16. Look at the value in the lower right corner
    • p = 4400
  17. Set other values to 0
    • x2 = 0
    • s1 = 0
    • s2 = 0

This leads to the solution of maximum profit of 4400EUR in the scenario of creating 200 pieces of product A, 0 pieces of product B, 300 pieces of product C, with both slack assembly and slack fabrication time being 0.
