Vitamin-Example

In the table above, there are multiple things to see:
  1. The RDA (Recommended Daily Allowance) of various vitamins
    • representing the recommended daily dosage for healthy vitamin levels
  2. The different types of vitamins, which the example is dealing with
  3. The amount of vitamins (in mg/kg) in 4 different types of food

The question is: “If we would want to cover these vitamins with only the aid of these four kinds of food, what would be the optimal way to do so?”
Now, how to proceed setting up the scenario? We may say, that the problem can be simplified expressed as “How much of what to eat?” and hence form the target formula:

x+y+z+q→optimize

Where “x, y, z, q” represent the amount of kilograms of each food consumed (multiplier is 1, since mg/1kg). But this needs to be transformed into a linear equation (with t representing the total amount of consumed food):

-x-y-z-q+t=0

The next step is to form the constraints, which are already represented by the table rows themselves. If the amounts of vitamins in each food type are to be taken and have the RDA as limitations, the constraints would look like:

0,9x+8y+0,4z+9,7q≤1,1

4,2x+2,7y+0,3z+4,9q≤1,4

0,7x+1,2y+1z+10q≤1,4

50x+20y+100z+10q≤80

1x+15y+5z+15q≤12

With these and the target function, we have constructed the Standard form, from where the next step is to create the Extended Standard form through the introduction of slack variables and fulfillment of the extended form rules mentioned in the other tutorials:

-x-y-z-q+t=0

0,9x+8y+0,4z+9,7q+s1=1,1

4,2x+2,7y+0,3z+4,9q+s2=1,4

0,7x+1,2y+1z+10q+s3=1,4

50x+20y+100z+10q+s4=80

1x+15y+5z+15q+s5=12

The example was further solved via Excel Solver, which offers a rather quick and simple solving tool for linear programming, described on the picture below (the Q column serves as an explanation for the fields to the right -> opt and the green ones; x,y,z,q are set to 0). With this setup, we may proceed to using the Solver add-in (make sure you have it implemented in MS Excel: File -> Options -> Add-ins -> Solver Add-in -> Go -> check Solver Add-in box and add), found in DATA -> Solver. Then, we add the Objective (in our case the opt cell), Variable cells (x,y,z,q cells) and Add constraints (green cells being less than red cells). Adding Constraints: After all has been setup, click Solve and OK in the next pop-up window. This presents the Solver solution of the problem: (All values rounded to 2 decimal places) These values bring us to the conclusion that the supposed optimum is achieved via consuming 0,24kg of food type x (Champ.), 0,08kg of food type y (Erbsen), 0,67kg of food type z (Äpfel) and 0kg of food type q (Soja), bringing us to the total amount of food consumed -> 0,99kg.
The slack variables represent the difference between the outcome vitamin value (green) and the RDA (red), being s1=0, s2=0, s3=0,47, s4=0 and s5=7,27.
