  1. What needs to be done first, is the setting of the constraints

  2. After the first necessary step, we begin shaping the Standard form

    1. creation is only allowed by setting constraints

  3. Followed by creating the Extended Standard Form

    1. formed via introduction of Slack/Surplus variables
    2. represented by fulfilling the following rules:
    3. --Non-negative constraints of all variables
    4. --All remaining constraints expressed as equalities
    5. --RHS vector being non-negative

  4. Forming the Canonic “Simplex” Tableau

  5. Performing pivot operations

    1. choosing the pivot column (via most negative number in the bottom row)
    2. choosing the pivot row (via dividing RHS by the row value and choosing the lowest result)
    3. making these two into the pivot (or pivoting point) and turning its value to 1
    4. making every value “above” and “below” to 0
    5. repeating this cycle until there are no negative numbers in the bottom row
